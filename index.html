🔥 Firebase Setup Instructions
Step 1: Create Firebase Project

Go to Firebase Console and create a new project

Step 2: Enable Realtime Database

In your Firebase project, go to "Realtime Database" and click "Create Database" (start in test mode)

Step 3: Get Firebase Config

Go to Project Settings > General > Your apps > Add app (web). Copy the config object.

Step 4: Update Database Rules

In Realtime Database > Rules, paste:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}
Step 5: Replace Firebase Config

Find the firebaseConfig object in the script section and replace with your own.

The system will work in offline mode until you configure Firebase.
